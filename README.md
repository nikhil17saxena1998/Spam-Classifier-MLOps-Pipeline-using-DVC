## Project: Spam Classifier MLOps Pipeline using DVC (Pipelines, Experiments, S3 Remote)

Built an end-to-end NLP spam classification pipeline (ingestion → preprocessing → TF-IDF features → model training → evaluation) in Python using scikit-learn + NLTK.

Implemented reproducible ML workflows with DVC pipelines and DVC Experiments + DVCLive for parameterized runs and metric tracking across iterations.

Enabled data/model versioning on AWS S3 via DVC remote, ensuring consistent reproducibility and easy collaboration through Git/GitHub.
